# RequestForConsulting

## 프로젝트 소개 💡

```
학교 상담 신청 페이지이다.
먼저 원하는 교수님을 선택 후 상담 내용, 날짜 및 시간을 선택하고, 확인 이메일을 입력해야한다.
시간은 오전 / 오후로 구분되고, 이미 선택된 날짜 또는 시간이 있다면 노란색으로 표시돼있다.
```
